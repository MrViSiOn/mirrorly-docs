AI-powered product visualization plugin for WooCommerce that generates realistic product images.

== Description ==

Mirrorly is a powerful WordPress plugin that integrates with WooCommerce to provide AI-powered product visualization capabilities. Generate realistic product images using advanced AI technology to enhance your online store's visual appeal and improve customer engagement.

**Key Features:**

* AI-powered image generation for products
* Seamless WooCommerce integration
* Easy-to-use admin interface
* Customizable image settings
* Bulk image processing
* API integration for advanced functionality

**Requirements:**

* WordPress 5.0 or higher
* WooCommerce plugin
* PHP 7.4 or higher
* Active internet connection for AI processing

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/mirrorly` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Navigate to WooCommerce > Mirrorly Settings to configure the plugin.
4. Enter your API credentials and configure your preferences.
5. Start generating AI-powered product images!
